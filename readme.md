# 🚀 MGPEB - Aurora Siger

Sistema de Gerenciamento de Pouso e Estabilização de Base (MGPEB) desenvolvido para simular a organização de módulos em uma colônia em Marte.

---

## 📌 Objetivo

Simular um sistema capaz de:

* Organizar módulos em fila de pouso
* Tomar decisões com base em regras lógicas
* Aplicar conceitos de programação em Python

---

## ⚙️ Funcionalidades

* Cadastro de módulos
* Fila de pouso
* Ordenação por prioridade
* Verificação de condições para pouso
* Simulação de autorização ou negação de pouso

---

## 🧠 Conceitos Utilizados

* Estruturas de dados (listas, fila e pilha)
* Lógica booleana (AND, OR)
* Estruturas condicionais (if / else)
* Algoritmos de ordenação simples

---

## 💻 Como executar

1. Baixe o arquivo `mgpeb.py`
2. Execute no Python:

```
python mgpeb.py
```
